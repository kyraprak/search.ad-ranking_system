# Search Ad Ranking System (Flask)

This project simulates how search engines rank advertisements using
bid value and relevance score, and presents the results in a
clean, interactive marketing-style dashboard built with Flask.

## Features
- Rank ads using a scoring formula (bid Ã— relevance)
- Filter ads by minimum bid and relevance
- Measure and display ranking performance time
- Interactive HTML table dashboard
- Visual highlighting of high, medium, and low performing ads
- Responsive, marketing-focused UI design

## UI & UX Design
The interface is designed to resemble a lightweight marketing dashboard:

- Color-coded performance scores for quick comparison
- Clear visual hierarchy to emphasize top-performing ads
- Clean, modern styling optimized for readability and usability
- Responsive layout suitable for desktop and tablet viewing


## Tech Stack
- Python
- Flask
- HTML / CSS
